# PokerForge — assets extraits de la maquette table

Ce dossier contient des PNG extraits de la maquette + des versions agrandies `_x3`, une palette, un guide JSON et un prompt Codex.

⚠️ Les assets proviennent d’une capture raster : ils ne remplacent pas les fichiers sources originaux. Pour du SVG/PNG haute résolution parfait, Codex devra redessiner les éléments à partir de ces références.

## Contenu principal
- `01_logo_pokerforge.png` : logo extrait.
- `05_poker_table_full.png` : table complète.
- `06_table_felt_texture_center.png` : texture feutre.
- `08_hero...` à `13_sb...` : sièges, avatars, cartes et stacks.
- `14_action_panel_right.png`, `16_action_buttons.png`, `17_bottom_controls.png` : panneaux et boutons.
- `palette_swatches.png` / `.svg` : palette.
- `style_guide.json` : spécification détaillée.
- `codex_prompt_recreation.md` : prompt prêt à coller dans Codex.

## Typographies
Exact impossible à certifier depuis une capture seule.
- UI : Inter / Manrope.
- Nombres et labels compacts : Rajdhani / Sora.
- Logo : style display italique condensé, à redessiner en SVG.

## Style cible
Premium dark / esport / sci-fi, panneaux verre fumé, table feutre vert avec double contour métal + or, glow cyan/or, cartes et avatars en profondeur.
